# Car Service — автосервис

Веб-сервис для учёта заказов на ремонт и обслуживание автомобилей.

## Тема

Сервис для автосервиса: учёт клиентов, автомобилей, механиков, запчастей и заказов на обслуживание.

## Основные сущности

- **Customer** — клиент (имя, email, телефон)
- **Vehicle** — автомобиль (номер, марка, год, владелец)
- **Mechanic** — механик (имя, специализация)
- **Part** — запчасть (название, цена, количество на складе)
- **ServiceOrder** — заказ на обслуживание (автомобиль, клиент, механик, запчасти, стоимость работ, статус)

## Операции сервиса

### CRUD
- Клиенты: создать, получить, обновить, удалить
- Автомобили: создать, получить, обновить, удалить
- Механики: создать, получить, обновить, удалить
- Запчасти: создать, получить, обновить, удалить
- Заказы: создать, получить, обновить, удалить

### Бизнес-операции
1. **Закрытие заказа** — `PUT /api/service-orders/{id}/close` (только если все работы выполнены)
2. **Расчёт суммы заказа** — `GET /api/service-orders/{id}/total` (запчасти + работа)
3. **Добавление запчастей к заказу** — `POST /api/service-orders/{id}/parts` — списание со склада в одной транзакции
4. **История заказов клиента** — `GET /api/service-orders/by-customer/{customerId}`
5. **Текущая загрузка механика** — `GET /api/service-orders/by-mechanic/{mechanicId}/open`

## Запуск

### Требования
- Java 17+
- PostgreSQL

### Настройка БД
1. Создайте базу: `CREATE DATABASE carservice;`
2. Скопируйте `.env.example` в `.env` и укажите подключение:

```
DB_URL=jdbc:postgresql://localhost:5432/carservice
DB_USERNAME=postgres
DB_PASSWORD=ваш_пароль
```

3. Запуск: `./mvnw spring-boot:run` (из папки `demo`)

Сервис доступен на http://localhost:8080

## Коллекция запросов

- `demo/requests/Car-Service-Postman.json` — коллекция для импорта в Postman (CRUD + 5 бизнес-операций).
- `demo/requests/car-service.http` — HTTP-запросы для VS Code REST Client, IntelliJ.

**Порядок проверки сценария:** создать клиента → механика → запчасти → автомобиль → заказ → выполнить бизнес-операции (расчёт суммы, добавление запчастей, закрытие заказа и т.д.).
